# MQ

## 为什么要用 MQ
1. 性能：通过异步处理提高系统性能（流量削峰）
2. 解耦：降低系统间耦合性
> 参考：[消息队列新手入门](https://mp.weixin.qq.com/s/MRHswsKoL0sm8ppd_gpF5Q)

## MQ 选择
1. RabbitMQ
2. Kafka
3. Spring Cloud Bus
4. MNS: Message Service 阿里云消息队列
